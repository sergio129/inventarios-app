# Manual de Atajos de Teclado - SaludDirecta POS

## Descripción General

SaludDirecta está optimizado para uso rápido con teclado, siguiendo estándares de supermercados y tiendas minoristas. Este manual te ayudará a maximizar tu productividad usando atajos de teclado.

---

## 1. NAVEGACIÓN PRINCIPAL (F1-F12)

| Atajo | Módulo | Descripción |
|-------|--------|-------------|
| **F1** | Ayuda | Abre el manual de atajos |
| **F2** | Ventas | Acceso directo al módulo de ventas (PRINCIPAL) |
| **F3** | Devoluciones | Módulo de devoluciones y cambios |
| **F4** | Inventario | Gestión de productos e inventario |
| **F5** | Reportes | Ver reportes y estadísticas de ventas |
| **F6** | Clientes | Gestión de clientes y datos |
| **F7** | Impresión | Configuración de impresora y recibos |
| **F8** | Configuración | Ajustes generales de la aplicación |
| **F12** | Cerrar Sesión | Cierra la sesión actual |

**Ejemplo de uso:**
- En cualquier pantalla, presiona **F2** para ir directo a Ventas
- Estando en Inventario, presiona **F3** para ir a Devoluciones

---

## 2. CONTROLES GENERALES

| Atajo | Acción | Descripción |
|-------|--------|-------------|
| **ESC** | Cancelar/Atrás | Sale del diálogo o regresa a pantalla anterior |
| **ENTER** | Confirmar/Buscar | Confirma una acción o busca un producto |
| **TAB** | Siguiente campo | Se mueve al siguiente campo en formularios |
| **SHIFT + TAB** | Campo anterior | Se mueve al campo anterior en formularios |

**Ejemplo de uso:**
- Presiona **TAB** para mover entre campos de un formulario
- Presiona **ESC** para cancelar un diálogo sin guardar

---

## 3. MÓDULO DE VENTAS (F2)

### 3.1 Búsqueda y Productos

| Atajo | Acción | Descripción |
|-------|--------|-------------|
| **ENTER** | Buscar | Busca el producto por código o nombre |
| **CTRL + F** | Búsqueda rápida | Abre el campo de búsqueda |
| **ESC** | Limpiar búsqueda | Limpia el campo de búsqueda |

### 3.2 Cantidad y Operaciones

| Atajo | Acción | Descripción |
|-------|--------|-------------|
| **0-9** + **ENTER** | Cantidad | Ingresa cantidad rápidamente (ej: 5 ENTER = 5 unidades) |
| **+** | Aumentar cantidad | Suma 1 a la cantidad actual |
| **-** | Disminuir cantidad | Resta 1 a la cantidad actual |
| **/** | Dividir | Divide la cantidad por 2 |

### 3.3 Acciones de Venta

| Atajo | Acción | Descripción |
|-------|--------|-------------|
| **C** | Calcular | Calcula el total de la venta |
| **P** | Pagar | Procesa el pago/abre formulario de pago |
| **D** | Descuento | Aplica descuento (global o por producto) |
| **E** | Cliente | Busca o selecciona cliente existente |
| **N** | Nuevo cliente | Crea un nuevo cliente rápidamente |
| **X** | Eliminar línea | Elimina el último producto agregado |
| **CTRL + X** | Limpiar carrito | Vacía todo el carrito (con confirmación) |

### 3.4 Métodos de Pago  

| Atajo | Acción | Descripción |
|-------|--------|-------------|
| **1** | Efectivo | Pago en efectivo |
| **2** | Tarjeta | Pago con tarjeta |
| **3** | Transferencia | Pago por transferencia |
| **4** | Cheque | Pago con cheque |

**Flujo típico de venta:**
1. Escanea/ingresa código → **ENTER**
2. Ingresa cantidad → **5** → **ENTER**
3. Otro producto → Código → **ENTER** → cantidad → **ENTER**
4. Termina → **C** (Calcular)
5. Cliente → **E** (busca cliente)
6. Pago → **P** → Elige método (1, 2, 3, 4)

## FLUJOS RÁPIDOS (Ejemplos)

### Venta Rápida (20 segundos)
```
1. Presiona F2 (Ventas)
2. Escanea código → ENTER
3. 2 (cantidad) → ENTER
4. Escanea otro código → ENTER
5. ENTER (cantidad 1)
6. C (Calcular)
7. E (Cliente) → selecciona
8. P (Pagar) → 1 (Efectivo)
9. Listo
```

### Devolución Rápida
```
1. Presiona F3 (Devoluciones)
2. ENTER (buscar ticket)
3. Ingresa número de ticket
4. ENTER
5. C (Cambio) o R (Reembolso)
6. P (Procesar)
```

### Búsqueda en Inventario
```
1. Presiona F4 (Inventario)
2. CTRL + F (búsqueda rápida)
3. Ingresa código o nombre
4. ENTER
5. E (Editar) si necesitas cambiar
```

---

## NOTAS IMPORTANTES

- ✅ Los atajos son **case-insensitive** (mayúsculas o minúsculas funcionan igual)
- ✅ ESC siempre cancela y regresa sin guardar cambios
- ✅ ENTER confirma acciones
- ✅ TAB navega entre campos
- ✅ Los números rápido se pueden escribir sin presionar caracteres especiales
- ⚠️ **CTRL + X (Limpiar carrito)** requiere confirmación
- ⚠️ **F12 (Cerrar sesión)** requiere confirmación
